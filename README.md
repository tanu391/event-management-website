# Event Management Website 🎉

## 📌 Project Overview
This is a responsive Event Management Website developed using HTML, CSS, and JavaScript. 
It allows users to view events, register online, and interact with an AI-powered chatbot.

## 🚀 Features
- Responsive Design (Mobile & Desktop)
- Hero Section with Background Image
- Countdown Timer
- Event Search Functionality
- Event Cards with Images
- Registration Form
- Gallery Page
- Admin Dashboard (Frontend UI)
- IBM Watson Chatbot Integration
- GitHub Pages Deployment

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- IBM Watson Assistant
- GitHub Pages

## 🤖 Chatbot Integration
IBM Watson Assistant is integrated to provide automated support for:
- Event details
- Registration queries
- Fees information
- Venue details
- Contact support
